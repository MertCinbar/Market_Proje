# Market Otomasyonu (C# WinForms)

Bu proje, C# WinForms kullanılarak geliştirilmiş bir market otomasyon sistemidir.

## Özellikler
- Yönetici, Kasiyer ve Müşteri rolleri
- Ürün fiyat güncelleme (Yönetici)
- Kasiyer ekranı (hızlı satış)
- Müşteri ekranı (sepet mantığı)
- Interface ve Abstract Class kullanımı
- Ortak veri yönetimi (static class)

## Kullanılan Yapılar
- Interface (ISatis)
- Abstract Class (BaseSatis)
- Static Class (Fiyatlar, OrtakAlan)
- OOP prensipleri

## Giriş Bilgileri
- Yönetici: Admin / 1234
- Kasiyer: Kasiyer / 4321
- Müşteri: Müşteri / 1111
